Если при socket справедливы все утверждения:

1) SOCK_CLOEXEC принадлежит sock_type,
2) SOCK_NONBLOCK принадлежит sock_type,
3) SOCK_DGRAM принадлежит sock_type,

то при обозначениях:

newFD — значение типа Nat, такое что newFD не принадлежит домену current.proc_fds и ∀ nn:Nat из nn < newFD следует nn ∈ dom current.proc_fds,
openfile — get_open_file(current,fd),
socket — get_sock_data(current,fd),

система должна:

  • Установить атрибуту fd_flags значения current.proc_fds по ключу newFD значение {FD_CLOEXEC}.
  • Установить атрибуту of_flags openfile значение {O_RDWR}.
  • Добавить O_NONBLOCK в openfile.of_flags.
  • Установить атрибуту socket_status socket значение UNCONNECTED.
  • Установить атрибуту of_op_type openfile значение SOCKET_TYPE.
  • Установить атрибуту socket_type socket значение SOCK_DGRAM.
  • Вернуть newFD.
